# Contributing

## Development setup

1. Clone the repository.
2. Place it in your WordPress plugins directory.
3. Activate the plugin and test in the block editor.

## Guidelines

- Keep the runtime lightweight.
- Preserve native Gutenberg UX patterns.
- Ensure new behavior respects reduced-motion preferences.
- Keep all bundled dependencies GPL-compatible.

## Pull requests

- Include a concise description of behavior changes.
- Include manual testing notes.
- Update `CHANGELOG.md` for user-facing changes.
